12. **Semantic variable-aliasing fork (CLD)** *(NEW — SenseRun #10; measurement-validity, arguably Meadows #5 STRUCTURAL → August + Ember, NOT self-applicable).* The harness (#5) deliberately did **not** map a model's free-form variable names onto oracle-canonical names — a hand alias map leaks the oracle and inflates scores; the honest scope was strong elicitation + the scorers' existing synonym tolerance. Whether to go further is a real fork: **(a)** do nothing (accept the scorers' synonym tolerance as the ceiling; residual mismatch is a true model property — current posture); **(b)** a *published, item-independent* alias map (general SD vocabulary only, never item-specific) — bounded leak risk; **(c)** embedding/similarity match scored as partial — softest, hardest to certify. Needs the first live CLD run's mismatch data to choose; held open, not averaged away.
